# python基础

## 基本

+ 输入输出  
输出：print([string])  
输入: x=input([string])

+ 运算符  
加+，减-，乘*，除/  
取余%，** 幂运算(3**2意为3的2次方)  
大于>,小于<,等于==,不等于!=，小于等于<=,大于等于>=  
与&,或|,非!

+ 注释  
单行：#  
多行：''' 

## 数据类型

+ 数字  
int,float,complex复数

+ 字符串  
str
len()长度,find()查找,replace()替换，slpit()切割

## 条件控制

+ 判断  
```
if true:
    print("true")
elif true:
    print("else if true")
else:
    print("false")
```

+ while  
```
while true:
    print("true")
```

+ for  
```
# 打印数字从0到9
for i in range(0,10,1):
    print(i)
```
